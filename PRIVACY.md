# Privacy Policy

agent-notes is a local development tool. It does not collect, store, or transmit any personal data.

## What runs locally

- All agents and skills execute within Claude Code on your machine
- Claude Code handles all API calls to Anthropic directly — agent-notes has no visibility into these calls
- Obsidian memory files (if configured) are written to your local vault and never leave your machine
- Configuration files are stored in `~/.config/agent-notes/` on your machine

## What we do not do

- No analytics or telemetry
- No data collection of any kind
- No external network requests from agent-notes itself
- No account required
